```
.onTapGasture 
``` 
**添加一个在此视图识别出轻击手势时执行的动作。**

example： 

```
Image(systemName: self.UserData.Todolist[index].isChecked ? "checkmark.square.fill" : "square")
                 .imageScale(.large)
                 .padding(.trailing)
                 **.onTapGesture {
                     self.isChecked.toggle()** // 制作一个按钮，如果点击这个就翻转
             }
             ```
